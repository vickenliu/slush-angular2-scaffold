## Angular2 Hydrane

## Install & Development
```bash
git clone https://github.com/vickenliu/hydraneVicken
cd hydraneVicken

# Install dependencies
npm install

# run and watch for changes in .ts files
npm start

# Applciation url: http://localhost:8080
```

## Deployment

```bash
npm run build
```

upload files in 'dist' folder to the server
